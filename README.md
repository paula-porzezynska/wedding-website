**> _ENG version_ <**
### Gallery on the website 
To display gallery properly, you need to add some images to the following folders:
> ./images/img
> ./images/thumbnails

with name:
> img_xxx.jpg (range of values: 001-591)

### Link to the website
You can visit website on: [pobieramysie.pm](http://pobieramysie.pm)
